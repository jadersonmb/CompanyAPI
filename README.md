API category que tem os seguintes endpoits.

  - Save
  - Delete
  - DeleteAll
  - FindById
  - ListAll

### Tech
  * [Java] - Backend with Java 11

### Installation

Dillinger requires [Java](https://www.java.com/en/download/) v11+ to run.

Install the dependencies and devDependencies and start the server.

```sh
Maven
Lombok
PostgreSQL
H2
```
